# Machine Learning with PyTorch and Scikit-Learn Study Repo

This repository documents my study of *Machine Learning with PyTorch and Scikit-Learn*.

It includes:

- chapter notes
- implementation exercises
- experiments and visualizations
- mathematical intuition and explanations
- small extensions beyond the original material

The goal of this repo is to strengthen my machine learning foundations while practicing clean implementation and structured experimentation.

---

## Goals

- Build strong intuition for core machine learning concepts
- Improve fluency with PyTorch and scikit-learn
- Practice implementing concepts instead of only reading them
- Develop the habit of running experiments and reflecting on results
- Create a structured public study log that can later inspire standalone projects

---

## Repository Structure

```text
chapters/       Chapter-by-chapter notes, code, and experiments
notes/          General notes and glossary
assets/         Figures, diagrams, and images used in notes
mini_projects/  Larger project ideas spun out from the book
```

---

## Chapter Workflow

For each chapter, I aim to include:

1. A short written summary of the key ideas
2. Important formulas, concepts, or implementation details
3. A cleaned notebook or Python script
4. At least one small experiment or extension of my own
5. A few reflections

---

## What is Original vs Adapted

Some baseline ideas or code patterns may be adapted from the book's official materials.

This repository is primarily a personal study and implementation log. Any adapted material is used for learning purposes, and I aim to add my own:

- explanations
- code cleanup
- experiments
- comparisons
- reflections

---

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Then open the relevant chapter notebook or run the scripts inside the chapter folders.
